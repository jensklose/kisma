tag()

tag(string $tag, bool $isKey, string $strip) : string

Given a simple name, clean it up to a Kisma standard, camel-cased, format.

Periods should be used to separate namespaces. Underscores should be used to separate identifier words

Examples:

  Class Name:            kisma.aspects.event_handling => \Kisma\Aspects\EventHandling
  Array Key:            my_event => MyEvent

Parameters

string $tag
bool $isKey

If true, the $tag will be neutralized
string $strip

If provided, it's value is removed from item before it's neutralized.

                                Example: "REQUEST_URI" would be "URI" with $strip = "REQUEST_"

string
